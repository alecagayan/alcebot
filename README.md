## How to start and run
1. Paste main.py into your text editor
2. Replace 'token' with your bot token
3. Replace 'owm_key' with your OpenWeatherMap API key
4. Run the program

## Necessary Packages
1. discord
2. random
3. textblob
4. asyncio
5. re
6. logging
7. pyowm
8. time
